# RateThatFilm
 ## RTF is a React based web application for reviewing movies
 
### Features -
 -Conveniently Search from Movies
 -Add & edit your own reviews

### Tech Stack -
-ReactsJS : Used routes for navigation

### Future Updates -
-Database connection
-User authentication
-OMDb API integration for fetching movie details
